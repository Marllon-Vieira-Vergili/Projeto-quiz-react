# Projeto Quiz - React

Quiz interativo desenvolvido em React para testar conhecimentos.

## Tecnologias utilizadas

- React
- JavaScript
- Frameworks React
- CSS
- Vite (para build)

## Como rodar

```bash
git clone https://github.com/Marllon-Vieira-Vergili/Projeto-quiz-react.git
cd Projeto-quiz-react
npm install
npm run dev

Acesse: http://localhost:5173

Projeto desenvolvido para portfólio e prática com React.
```
